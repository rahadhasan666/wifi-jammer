## Run Locally

Clone the project

```bash
  git clone https://github.com/rahadhasan666/wifi-jammer.git
```

Go to the project directory

```bash
  cd wifi-jammer
```

Exicute

```bash
 sudo chmod +x jamer.sh
```

Start Jammer

```bash
  sudo ./jamer.sh
```
